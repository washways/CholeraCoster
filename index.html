How to Use This Calculator

  • Select a country from the dropdown at the top (defaults to Malawi)
  • Wait for data to load — you'll see a loading spinner while fetching World Bank API data
  • Blue-highlighted fields API are automatically filled from World Bank data

  • Export or print your analysis using buttons at the bottom

How it works

This calculator uses a combination of user input and publicly available data to project the costs and benefits of cholera prevention strategies in Malawi.

  • Population data (highlighted in blue) is automatically loaded from the World Bank API when the page loads. Additional indicators such as GDP per capita, sanitation coverage, health expenditure and stunting are also fetched and shown in the panel.
  • The model computes a phased ramp-up of WASH interventions and OCV campaigns over the chosen timeline.
  • Health outcomes are derived from baseline cholera rates, coverage targets, and efficacy assumptions; cases and deaths averted flow through a rich set of economic formulas to calculate benefits including emergency response savings, diarrhea treatment avoided, productivity gains, funeral/tourism impacts and more.
  • Costs include WASH capital and O&M, vaccination campaigns, surveillance, case management and a user‑defined "other program" per‑capita line item.
  • All formulas run in your browser using JavaScript; no data is ever sent to a server.

Fields pre-filled from the API are visually distinguished so you can easily identify automatic inputs.
