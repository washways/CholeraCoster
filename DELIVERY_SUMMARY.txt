**Q: Do I need to install anything?**
A: No! Python built-in HTTP server handles it. Just click run.bat.

**Q: Can I share this with colleagues?**
A: Yes! Send the entire CholeraCoster folder. They can run locally.

**Q: Can I modify the calculations?**
A: Yes! Edit calculator.js with any text editor. See README.md for formulas.

**Q: Will it work offline?**
A: Yes! After one load, works without internet (except World Bank API data).

**Q: Is my data private?**
A: 100%! All calculations run in your browser. No data sent anywhere.

**Q: How accurate is it?**
A: Accurate as your inputs. Based on WHO methodology. See README.md for details.
